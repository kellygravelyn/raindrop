Raindrop is my personal color theme I use for all of my software when I can. I use it for code editing as well as in my terminal. I made Raindrop because I wanted a light theme that felt calm and simple.

A key feature of Raindrop, as far as the Sublime Text color scheme goes, is incredibly minimal use of selectors. I find that the majority of code is perfectly fine being rendered in a dark color on a light color. I apply colors judiciously only where I feel like it makes my experience better.

Also critically I am not a designer. I don't have a background in color theory. There is probably a lot to improve here and I do over time as I experience these colors in my daily life. If you have suggestions, please feel free to open a GitHub issue to discuss it.
